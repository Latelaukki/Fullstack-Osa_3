# Fullstack-Osa_3

Fullstack kurssin osa 3 tehtävät
<br>

## Muistiinpanoja

* Nodemon päivittää serverin muutokset 
<br>

## Komentoja

* Backendin luonti ilman Reactia:
<br> 
`npm init`
<br>

* Serverin ajaminen Nodemonilla: 
<br>
`npm run dev`
<br>

* Serverin ajaminen normaalisti:
<br>
`npm start`
<br>
